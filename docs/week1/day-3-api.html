今日验收

产出物API 契约表：方法、路径、请求、响应、状态码、错误行为、重试说明。
必须掌握POST 不天然幂等。GET 跳转读取状态并返回 HTTP 302。链接可能过期或变化时，301 有缓存风险。
安全边界不要信任请求体里的 user_id；身份只能来自认证上下文。
修复规则接口名如果像动词动作，改成资源名；如果错误码影响客户端行为，就必须写出来。
