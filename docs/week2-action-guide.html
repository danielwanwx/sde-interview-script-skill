本周目标

把读扩展、缓存、Redis、consistent hashing 和 rate limiter 打成一组基础设施能力。

必须掌握
  • Caching、Redis、consistent hashing、rate limiter algorithms。
  • Distributed Cache 和 Rate Limiter 至少各能完整讲一遍。
  • 故障路径必须包含 hot key、cache outage、Redis failover。
本周不展开
  • 消息队列深挖、stream processing、复杂多 region 配额。
  • 业务风控和 abuse detection 只作为扩展。

时间线

每天按照学习源包、今日验收、算法模块执行。拓展阅读只在主任务完成后使用。
